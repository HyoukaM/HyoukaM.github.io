出来以后简单的学了一下vue3，发现通过setup函数解构出去的变量不会被监听到🧐
```javascript
//不会响应
<template>
    <div>
        {{name}}
    </div>
</template>
<script>
    export default {
        setup() {
            const state = {
                name: 'this is vue3'
            };
            return {...state}
        }   
    }
</script>

//会响应
<template>
    <div>
        {{state.name}}
    </div>
</template>
<script>
    export default {
        setup() {
            const state = {
                 name: 'this is vue3'
            };
        return {state}
    }
}
</script>
```
也是第一次试vue3，还是需要慢慢来的。
